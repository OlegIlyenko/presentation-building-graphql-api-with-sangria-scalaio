Common Issues

  • Over-fetching

      /products?field=description&
      /products?field=variants[*].price
  • Under-fetching
    • /products?expand=productType&
      /products?expand=variants[*].price.taxRate
  • API changes and evolution
    • Versioning
    • Deprecation
    • Maintenance

GraphQL Approach

GraphQL Query


{
  "data": {
    "product": {
      "name": "Delicious Cake",
      "description": "Just taste it!"

      "picture": {
        "width": 150,
        "height": 150,
        "url": "http://..."
      }
    }
  }
}

GraphQL Query


{

     product   {
       name
       description

       picture   {
         width
         height
         url
      }
    }

}
